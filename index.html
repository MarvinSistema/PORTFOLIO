GitHub Projects

Here are some of my projects hosted on GitHub:

  • WEB-DIA - Este proyecto es una innovadora solución de automatización de viajes diseñada para transformar y optimizar los procesos logísticos de transporte. Con el objetivo de reducir significativamente los tiempos operativos, el sistema utiliza una interfaz web intuitiva que permite a los usuarios visualizar en tiempo real los recursos disponibles, incluyendo la flota de vehículos y conductores. Además, el programa ofrece una asignación automática y precisa de los viajes, garantizando una logística eficiente y maximizando la utilización de recursos.
